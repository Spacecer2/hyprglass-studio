# Wallust Integration

Hyprglass Studio extracts accent colors from your wallpaper using [wallust](https://github.com/anufrievroman/wallust), keeping the glass tint synchronized with your desktop theme automatically.

## What is Wallust

Wallust analyzes your wallpaper and extracts a color palette. It generates terminal-style color schemes (`$color0`–`$color15`) plus semantic variables like `$background` and `$foreground`, making it easy to derive consistent theme colors from any image.

## Available Color Variables

| Variable | Description |
|---|---|
| `$color0`–`$color15` | 16-color palette extracted from the wallpaper |
| `$background` | Dominant background color |
| `$foreground` | Dominant foreground/text color |

Hyprglass primarily uses `$color12` as the accent and `$background` to calibrate brightness and edge contrast.

## Template System

### Template Location

```
~/.config/wallust/templates/colors-hyprglass.conf
```

Wallust processes this template on every wallpaper change, substituting variables and writing the result to a generated config that Hyprglass includes.

### Template Content

```conf
# Auto-generated by wallust – do not edit directly.
# Customize via ~/.config/hyprglass/hyprglass.conf instead.

$accent       = rgba({color12}cc)
$tint         = rgba({color12}66)
$bg_lum       = luminance({background})

# Brightness: darker wallpapers get a lighter glass, lighter wallpapers get a darker glass
$brightness   = if($bg_lum < 0.3, 0.85, 0.45)

# Edge thickness: higher contrast backgrounds need thicker borders
$edge_thick   = if($bg_lum < 0.3, 2, 1)

# Contrast tint for readability
$contrast_tint = rgba({background}aa)
```

### Output

Wallust writes the rendered template to:

```
~/.config/hyprglass/wallust-generated.conf
```

This file is sourced by `Hyprglass.conf` at startup and on reload.

## Color Mapping

Hyprglass translates wallust variables into glass properties:

| Glass Property | Source | Logic |
|---|---|---|
| `tint_color` | `$color12` | Primary accent, alpha channel set to `66` (~40% opacity) for the glass tint |
| `brightness` | `$background` luminance | Dark backgrounds (`lum < 0.3`) → `0.85` (lighter glass). Light backgrounds → `0.45` (darker glass) |
| `edge_thickness` | `$background` luminance | Low luminance → `2px` border for visibility. High luminance → `1px` |

The luminance check ensures the glass effect remains visible regardless of wallpaper brightness.

## Trigger Mechanism

Color sync is driven by a single script:

```
WallustSwww.sh
```

This script runs whenever the wallpaper changes (typically triggered by `swww` or your wallpaper manager):

1. **Wallpaper change detected** → `WallustSwww.sh` executes
2. **Wallust processes** `colors-hyprglass.conf` → writes `wallust-generated.conf`
3. **`hyprctl reload`** applies the new config to all Hyprland windows

The full cycle happens in under a second on most systems.

## Customization

### Override Auto-Generated Values

Edit `~/.config/hyprglass/hyprglass.conf` to set values manually. Manual settings take precedence over wallust-generated ones:

```conf
# Force a specific tint color instead of auto-extraction
tint_color = rgba(ff6b6bcc)

# Override brightness (0.0 = darkest, 1.0 = brightest)
brightness = 0.7

# Fixed edge thickness in pixels
edge_thickness = 2
```

### Manual vs. Auto Tint

| Mode | Behavior |
|---|---|
| **Auto** (default) | `$color12` from wallust drives `tint_color`. Changes with every wallpaper. |
| **Manual** | Set `tint_color` in `hyprglass.conf`. Wallust value is ignored. |

To switch back to auto, remove or comment out the `tint_color` line in your config.

### Using a Different Accent Color

If `$color12` doesn't suit your palette, edit the template directly:

```conf
# Use color13 instead of color12
$tint = rgba({color13}66)
```

Wallust will pick up the change on the next wallpaper update.
