# HyprGlass Studio Screenshots

This directory contains screenshots showcasing the HyprGlass Studio project.

## Required Screenshots

| Filename | Description |
|----------|-------------|
| `terminal-glass.png` | Terminal window with glass transparency effect |
| `browser-glass.png` | Browser window with glass transparency effect |
| `studio-ui.png` | HyprGlass Studio web interface |
| `profile-menu.png` | Rofi profile selector menu |
| `gaming-mode.png` | Gaming mode with glass effect disabled |
| `wallust-sync.png` | Wallpaper change showing tint color update |
| `comparison.png` | Before/after comparison of glass effect |

## Usage

Add screenshots to the root `README.md` using relative paths:

```markdown
![Terminal Glass](screenshots/terminal-glass.png)
```

## Guidelines

- Use PNG format for all screenshots
- Keep dimensions reasonable (1920x1080 or smaller recommended)
- Name files using lowercase kebab-case
